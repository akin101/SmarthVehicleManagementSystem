 Project Overview

This is a simple Java console-based vehicle management system. The program reads vehicle data, processes tax calculations, handles invalid inputs, and displays filtered results based on specific conditions.

The main goal of this project is to practice object-oriented programming, file/data processing, and basic error handling in Java.

 Features
Vehicle data loading and validation
Emission value check (0–500 range control)
Error handling for missing or incorrect data
Tax calculation for each vehicle
Filtering high-tax vehicles
Displaying all vehicles in system
Console-based output logging
